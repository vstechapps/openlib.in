openlib.in

Build Faster. Ship Smarter.

openlib is a modern ecosystem of high-performance, lightweight Node.js libraries designed to help developers build scalable applications with minimal dependencies.

npx openlib create api-app

Why openlib?

⚡ Performance First

Built for speed with minimal overhead and optimized execution.

🧩 Modular Design

Use only what you need. Every package is independent and composable.

🚫 Less Dependencies

Reduce bloat and improve reliability with lightweight libraries.

🧠 Developer Friendly

Clean APIs and consistent patterns across all packages.

Core Packages

@openlib/api

Fast and minimal API framework for Node.js applications.

@openlib/db

Unified interface with adapter-based support for multiple databases.

@openlib/ui

Lightweight UI primitives for building modern interfaces.

@openlib/apitest

Simple and powerful API testing utilities.

Vision

openlib aims to become the go-to ecosystem for developers who want full control, maximum performance, and minimal complexity—without relying on heavy frameworks.
